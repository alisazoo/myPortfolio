Project

MYGE - Moyogae -

standalone application

  • Language: Java
  • IDE: IntelliJ
  • VCS: git

MYGE stands for "Moyo-gae", which means changing interior arrangement in Japanese. This application helps you to think about how to allocate the pieces of furniture in your room visually. You can set the sizes of the floor and items in your room and move them freely. Just measure the size of your room and furniture, then enjoy "Moyo-gae"!
